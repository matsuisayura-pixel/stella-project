## 5つの診断質問

### Q1：今のエネルギー状態
```
今のあなたのエネルギーは、どちらに近いですか？

A）外に向いている（何かしたい・動きたい）
B）内に向いている（休みたい・ひとりになりたい）
```

### Q2：今の悩みのタイプ
```
今、最も気になっていることは？

A）お金・仕事・目標
B）人間関係・恋愛・家族
C）自分自身（自信・使命・迷い）
D）特になし・穏やか
```

### Q3：行動パターン
```
何か問題があったとき、あなたはどうしますか？

A）すぐ動いて解決しようとする
B）しばらく考えてから動く
C）誰かに相談する
D）自然に任せる
```

### Q4：今の「季節感」
```
今の自分を季節に例えると？

A）春（新しいことが始まりそう）
B）夏（全力で動いている）
C）秋（収穫・振り返りの時期）
D）冬（休息・準備の時期）
```

### Q5：今一番欲しいもの
```
今、あなたが一番欲しいものは？

A）背中を押してくれる言葉
B）「休んでいい」という許可
C）これからの道筋・方向性
D）ただ、誰かに話を聴いてほしい
```

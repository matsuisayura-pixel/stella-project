# 知識ベース 素材登録ルール

## 概要
10年分のスピリチュアル素材を整理・検索しやすい形に保管するルール。

---

## フォルダ構成

```
knowledge-base/
├── videos/     ← 動画素材（文字起こしテキスト）
├── pdfs/       ← PDF素材（テキスト抽出済み）
└── texts/      ← テキスト素材
```

---

## 動画素材の登録方法（videos/）

### ファイル名ルール
```
YYYY-MM_タイトル.md
例：2023-06_引き寄せの本質とは.md
```

### ファイル内容のフォーマット
```markdown
# タイトル

## 基本情報
- 収録日：YYYY-MM-DD
- 尺：○分
- テーマタグ：#引き寄せ #感情 #行動

## 文字起こし（全文 or 要約）
〜ここに文字起こしを貼る〜

## 名言・エッセンス抽出（コンテンツに使える一文）
- 「〜〜〜」
- 「〜〜〜」

## ステラ変換メモ（口調変換後のフレーズ）
- 原文：「〜〜〜」
  変換：「〜〜〜なのです」
```

---

## PDF素材の登録方法（pdfs/）

### ファイル名ルール
```
YYYY_タイトル.md（元PDFのテキストを抽出してmd保存）
例：2022_感情解放メソッド.md
```

---

## テキスト素材の登録方法（texts/）

### ファイル名ルール
```
YYYY-MM_タイトル.md
例：2024-01_冬の季節の過ごし方.md
```

---

## タグ一覧（共通）

コンテンツ検索に使うタグ。登録時に必ず付ける。

```
#引き寄せ  #感情  #行動  #受容  #癒し  #自己愛
#人間関係  #お金  #仕事  #使命  #直感  #恐れ
#季節（春）#季節（夏）#季節（秋）#季節（冬）
#初心者向け  #上級者向け
```

---

## 今すぐできる素材登録の手順

1. 動画 → 手動で要点をメモして `videos/` に保存
2. PDF → テキストをコピーして `pdfs/` に保存
3. テキスト → そのまま `texts/` に保存
4. タグを付ける

（Phase 2 以降：Whisper等で文字起こし自動化予定）

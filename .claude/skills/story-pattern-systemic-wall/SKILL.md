# story-pattern-systemic-wall
## 物語YOUTUBE台本パターン②「組織の壁型」

---

## このパターンが解決する問題

A型テンプレートの弱点：「漫画的な悪役」が視聴者のリアリティ感覚を壊す。
深夜に職場まで嫌がらせに来る不動産担当者、子供に施設を示唆する課長——これは現実に起こらない。

**100点に必要な核心：**
> 最も残酷な壁は「悪意のない人間」が「規則通りに」作る。
> 担当者が謝りながら致命的なことをする——その乖離が現代の視聴者に最も刺さる。

---

## パターン構造（全9幕）

### 幕1：冒頭フック（制度の言葉で始める）
- **形式**：丁寧な敬語・マニュアル語で語られる絶望
- 暴言より「申し訳ございませんが」で始まる冷たさのほうが深い
- 例：
  ```
  「川村様、規定上これ以上のご猶予はお受けできません。
   誠に恐縮ですが、来週金曜までにご対応いただけますよう、よろしくお願いいたします」
  担当者は深々と頭を下げた。笑顔だった。
  ```

### 幕2：主人公の背景（なぜこの制度に挟まれているか）
- 主人公が「制度の外に落ちた」経緯を具体的に示す
- 落ちた原因は主人公の失敗ではない（病気・解雇・配偶者の死など）
- **重要**：「ここにくるまでの努力」を数値で示す
  - 例：「ハローワークに37回通った」「求人に84社応募した」

### 幕3：屈辱①「規則通りの拒絶」
- **悪役の描き方の絶対ルール**：
  ```
  NG: 「貧乏人は消えろ」（漫画的悪意）
  OK: 「お気持ちはわかります。ただ、弊社の規定上……」
      （事務的共感 + 規則による拒絶の組み合わせ）
  ```
- 担当者のセリフに「私も（上から）言われているので」を入れる
- 制度の「上流」が見えるが、誰も責任を取らない構造を示す

### 幕4：善行の機会（制度の外で人間と出会う）
- 主人公が「制度に疲弊した状態」で、制度の外の人物と出会う
- 相手は主人公に何も求めない——ただそこにいるだけ
- **善行の描き方**：
  - 大きな決断ではなく、「なんとなく」の行動として描く
  - 「なぜ助けたか」を主人公自身が説明できない
  - 後から「なぜあのとき」と自問する

### 幕5：屈辱②「善行のコストが現実に来る」
- 善行によって生じた具体的な損失を描く
- **現実的な数字を使う**：
  - 「1,980円が消えた。残高は820円」
  - 「3時間睡眠が2時間になった」
- 制度の壁が次の局面に入る（催告書・差押予告など、現実の法的プロセス）

### 幕6：屈辱③「人間的な訴えが制度に弾かれる」
- 主人公が「人として」訴えようとする場面
- 相手は「人として」聞こうとするが、「担当者として」拒絶する
- **最も効果的な表現**：
  ```
  担当者の目に、一瞬だけ何かが揺れた。
  しかし彼は書類を閉じ、立ち上がった。
  「大変申し訳ございません。私どもでは判断できかねます」
  ```
- この「揺れた目」が、制度の残酷さを最大化する

### 幕7：最絶望ポイント（制度が人を押し潰す寸前）
- 法的な最終通告が来る（内容証明・明渡訴訟の予告）
- 主人公が「制度と戦う」ことを一瞬考えるが、弁護士費用や時間を計算して諦める
- **このリアルな「諦め」の計算が視聴者のリアリティを高める**

### 幕7.5：もう一度だけ（ツイスト前の小さな善行）
- 最絶望の直後、主人公が「意識せず」もう一度だけ誰かを助ける
- 今度は金銭ではなく、時間・言葉・体力の提供
- これが「ツイストを受け取る資格の証明」になる

### 幕8：ツイスト（制度が逆転する）
- **構造の逆転**：主人公を苦しめた「制度・組織」が、今度は主人公を守る側に回る
- ツイストは「大物の登場」より「制度の論理が逆方向に働く」ことで起こす
- 例：
  ```
  主人公を苦しめていた「規則」が、実は主人公を守る規則でもあった
  それを知る人物が現れ、制度を主人公のために動かす
  ```
- **伏線の仕込み方**：
  - 悪役の「うちはバックに〇〇がいる」という一言を、ツイストの反転に使う
  - 最初の登場時に「〇〇の話」を1回だけ、さりげなく入れる

### 幕8.5：制裁（静かな形で）
- 派手な怒号・即日解雇より「静寂の3秒間」を使う
- 加害者が「自分が何をしたか」を理解した瞬間の描写
- その後の行動：背中が丸まる・声が出なくなる・謝罪の言葉が出ない
- **加害者を徹底的に叩かない**：それが却って格調を上げる

### 幕9：恩送り（丁寧に・400字以上）
- 主人公が初めて「制度の外で」誰かを助ける
- 今度は制度の言葉を使わず、人間として関わる
- **必須要素**：
  - 主人公が躊躇する（「また、迷っている」）
  - それでも動く（「でも足が動いた」）
  - 相手の表情より、主人公の「手の感触」を描く
- **エンディングの最後の一文**：制度ではなく「人」を見ている主人公の視点で終わる

## キャラクター設計

### 拮抗役（組織の担当者）
```
名前: 田中 / 山本 など平凡な名前
外見: グレーのスーツ・ネームプレート・清潔感
セリフパターン:
  - 「規定上〜」
  - 「弊社の判断ではなく〜」
  - 「お気持ちはわかります、しかし〜」
  - 「上から指示がありまして〜」
人間的な一面（必ず1つ入れる）:
  - 主人公が帰った後に溜息をつく
  - 廊下で「俺だってやりたくない」と独り言
  - 主人公の子供の話を聞いたとき、一瞬だけ書類を持つ手が止まる
```

### 組織の「上流」（見えない悪）
- 直接登場させない
- 担当者が「〇〇さんに言われているので」と口にするだけ
- この「見えない悪」がツイストで「見える」ようになる

## リアリティ・チェックリスト

- [ ] 制度の描写が実際の日本の法律・手続きに沿っているか
  - 家賃滞納の明渡しには最短でも2〜3ヶ月と裁判所手続きが必要
  - 解雇通告は30日前の予告が原則
- [ ] 悪役のセリフに「規則」「規定」「指示」が含まれているか
- [ ] 担当者に「人間的な揺れ」が1回あるか
- [ ] 善行の金額・内容が現実的か
- [ ] 奨学金・支援金の金額が現実的か（月3〜8万円）
- [ ] ツイストの伏線が1回のみか
- [ ] 恩送りシーンが400字以上か

## 使用方法

```
/story-pattern-systemic-wall テーマ:[テーマ] 主人公:[主人公設定] 舞台:[舞台設定]
```

例：
```
/story-pattern-systemic-wall
テーマ: 医療費が払えず治療を断られた母親が、病院の清掃員に助けられる
主人公: 50代女性・元正社員・夫の死後に経済的困窮・持病持ち
舞台: 都内病院・市役所・古いマンション
```

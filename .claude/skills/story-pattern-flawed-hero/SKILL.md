# story-pattern-flawed-hero
## 物語YOUTUBE台本パターン④「傷ある主人公型」

---

## このパターンが解決する問題

A型テンプレートの弱点：主人公が「完璧な善人」になる。
迷わず全財産を渡し、笑顔で見送る——これは感情移入の障壁になる。

**100点に必要な核心：**
> 欠点を持った人間が善行をするとき、その善行は「人間のベストの姿」になる。
> 「あなたもこんなことがある」という主人公の弱さが、「あなたもできる」という希望になる。

---

## 主人公の「傷」設計（必ず1つ選ぶ）

このパターンの最大の特徴は主人公の「傷」。以下から1つ以上を選び、物語に組み込む：

### 傷A：怒り型
```
特徴: 過去の不公平な経験から、他人を信用しにくくなっている
     「なぜ自分ばかり」という怒りを内側に持っている
物語での使い方:
  - 善行の対象を見たとき、「また自分が損をするのか」と思う
  - 善行をしながら、心の中で怒っている
  - その怒りのまま動いた自分に、後から驚く
NGライン: 怒りを完全に消してはいけない（消えるのは「成長」ではなく「都合」）
```

### 傷B：嫉妬型
```
特徴: 恵まれた人間を見ると、羨ましさと惨めさが混在する
物語での使い方:
  - 善行の対象が「かつての自分より恵まれた状況の人」であることもある
  - 「なぜあなたを助けなければならないのか」という感情
  - それでも助ける——その矛盾した行動
```

### 傷C：あきらめ型
```
特徴: 一度大きな失敗や裏切りを経験し、他人への関与を避けるようになった
物語での使い方:
  - 「関わっても無駄だ」という経験則から来る回避
  - 善行の対象を前に「また失敗するかもしれない」と思う
  - その怖さを抱えたまま動く
```

### 傷D：自己嫌悪型
```
特徴: 自分を大切にできない。他人を助けることで自己存在を確認しようとする
物語での使い方:
  - 善行の動機が純粋でないことを主人公自身が知っている
  - 「私が助けたいのか、助けることで自分を保ちたいのか」という自問
  - その不純な動機のまま行動した結果が、純粋に誰かを救う
```

## パターン構造（全9幕）

### 幕1：冒頭フック
- 主人公の「傷」を最初に示唆する
- 衝撃的な場面から始めつつ、主人公が「完璧な善人ではない」ことを最初に示す

### 幕2：主人公の「傷」の背景
- 傷がどこから来たのかを具体的に描く
- 過去の出来事（1〜2シーン）
- **重要**：傷は主人公の「弱さ」ではなく「経験の証拠」として描く
- 視聴者が「この人はそういう人なんだ」と理解できる程度

### 幕3：現在の日常（傷とともに生きている状態）
- 傷による「癖・回避行動・思考パターン」を日常シーンに盛り込む
- 例（怒り型）：些細なことに心の中でイライラするが、表に出さない

### 幕4：屈辱①
- 主人公の傷に触れる形の屈辱
- 例（嫉妬型）：恵まれた人間から見下される

### 幕5：善行の機会（「傷」が最も邪魔をする場面）
- **このパターンの心臓部**
- 善行の対象が現れたとき、主人公の「傷」が最大の抵抗勢力になる

**必須の内面描写構成：**
```
[善行の対象が現れる]
 ↓
[主人公の傷が反応する]
 例（怒り型）: 「また私が割を食うのか」
 例（嫉妬型）: 「なんであなたを助けなければならない」
 例（あきらめ型）: 「関わっても何も変わらない」
 ↓
[それでも足が動く理由——理性ではなく「何か」]
 重要: この「何か」を明確に言語化しない
 「なぜか足が止まっていた」「気づいたら財布を出していた」
 ↓
[渡した後の複雑な感情]
 善行をしたのに「よかった」と思えない
 むしろ「なぜ私が」という感情がまだある
 その複雑さを丁寧に描く
```

### 幕6：屈辱②③
- 善行のコスト + 追加の打撃
- 主人公の傷が再び痛む場面

### 幕7：最絶望ポイント（傷が完全に開く）
- 主人公が「やっぱり自分が馬鹿だった」と思いかける
- 過去の失敗と現在の失敗が重なって見える
- **重要**：ここで「善行を後悔する」という感情を正直に書く
  - 「あの老人に渡さなければよかった」という思考が一瞬頭をよぎる
  - しかし次の瞬間、その思考を恥じる
  - この「後悔→自己嫌悪」の往復が人間らしさを作る

### 幕8：ツイスト
- 善行の受け手が実は「主人公の傷に関係する人物」だった——という展開も有効
  - 例：主人公が信頼できなくなった「業界の人間」が、実は最も主人公を評価していた
- または：主人公の「傷」の原因になった出来事を、恩人が知っていた
- **傷が報われるのではなく、傷があったからこそ気づけた何かが明らかになる**

### 幕9：制裁・恩送り
- **恩送りシーンの特別な設計**：
  - 次の善行の相手を前に、主人公の傷が「また」反応する
  - 「またやってしまいそうだ」という感覚
  - でも今度は「まあいいか」という軽さが少しある
  - 完全に癒えていない傷を抱えたまま、手を伸ばす
  - **これが成長の証明**（傷が消えるのではなく、傷と共存できるようになる）

---

## 「傷」の描写テクニック

### 身体に宿らせる
```
NG: 「彼女は人を信じることができなかった」（説明）
OK: 「誰かに近づかれると、肩に力が入る癖がついていた」（身体描写）
```

### セリフに滲ませる
```
NG: 「私は嫉妬深い人間なんです」（告白）
OK: 「あなたはいいですよね、と言いかけて、口を閉じた」（行動+中断）
```

### 傷と反する行動に使う
```
「人を信用しない彼女が、なぜか財布を開けていた」
この矛盾が物語の核になる
```

## リアリティ・チェックリスト

- [ ] 主人公の「傷」が具体的な過去の経験から来ているか
- [ ] 善行シーンで傷が「抵抗」として機能しているか（最低3行の内面描写）
- [ ] 善行の後に「複雑な感情」があるか（すっきりしない）
- [ ] 最絶望ポイントで「善行への後悔」が一瞬あるか
- [ ] 恩送りシーンで傷が「消えず」に「共存」しているか
- [ ] 傷の描写が説明文ではなく身体・行動・セリフで表現されているか
- [ ] 恩送りシーンが400字以上あるか

## 使用方法

```
/story-pattern-flawed-hero テーマ:[テーマ] 主人公:[主人公設定] 傷:[傷の種類] 舞台:[舞台設定]
```

例：
```
/story-pattern-flawed-hero
テーマ: 元詐欺師が更生後に、詐欺被害者を助けた
主人公: 35歳男性・元特殊詐欺グループ員・服役後に仮出所・過去への罪悪感と自己嫌悪
傷: 自己嫌悪型（自分を信じることができない）
舞台: 下町の居酒屋・警察署・元被害者の家
```

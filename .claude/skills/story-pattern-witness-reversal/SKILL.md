# story-pattern-witness-reversal
## 物語YOUTUBE台本パターン⑤「逆転の証人型」

---

## このパターンが解決する問題

A型テンプレートの弱点：ツイストが「冒頭から読める」。
老人の指輪・「藤原さんがバック」という言葉——これらは意図せずツイストを予告してしまう。

**100点に必要な核心：**
> ツイストの人物を「第三者の視点」から描くことで、読者はその人物の正体を全く推測できない。
> 証人は善行も悪事も「外から」見ている。その「見る目」がツイストの武器になる。

---

## パターンの基本構造

このパターンは「視点の設計」が全て。

```
通常のA型:
  主人公視点 → 老人が現れる → 老人の指輪に注目 → 読者「この人が大物」と予測

証人型:
  証人の視点 → 主人公と見知らぬ人物の関係を「外から」観察
  → 証人には老人が誰なのか全くわからない
  → ツイストが「証人にとっての驚き」として描かれる
  → 読者は証人と同じ情報量しか持てない → 予測不能
```

---

## 証人キャラクターの設計（最重要）

### 証人の条件
1. 主人公とは直接の関係がない（知り合いではない）
2. 物語の場所に「いる理由」が自然にある
3. 主人公の善行を「偶然」目撃する
4. 悪役の悪事も「偶然」目撃する
5. ツイストの人物の正体を「知っているが、主人公には言わない」役割

### 証人の種類（どれかを選ぶ）

**証人A：その場所の常連**
```
コンビニの常連客・銭湯の常連・駅のホームの常連
毎日そこにいるから、主人公のことをよく知っている
しかし主人公は証人の存在をほとんど意識していない
```

**証人B：同じ境遇にいる人物**
```
主人公と似た境遇（貧困・喪失・孤独）にある人物
互いに「仲間」として認識はしていないが、似た場所にいる
証人は主人公の善行を見て「自分にはできない」と思う
```

**証人C：別の立場から見ている人物**
```
警備員・店員・清掃員・タクシー運転手
その場所の「背景にいる人物」
いつもそこにいるが、誰も注目しない
この「見えない人物」が全てを見ていた
```

## パターン構造（全9幕）

### 幕1：証人の登場（冒頭から証人視点）
- **このパターン唯一の冒頭構成**：
  - 「証人」の視点から物語が始まる
  - 証人は今日も「その場所」にいる
  - 主人公を見かけたことがある（毎日見ている）
  - でも言葉を交わしたことはない

```
構成例：
「あの女の子が今夜もコンビニのレジに立っていた。
 俺はいつもの席に座り、いつものコーヒーを注文した。
 名前も知らない。でも、顔は知っている。
 いつも疲れた顔をしている」
```

### 幕2：証人が主人公を「観察」する
- 証人は主人公のことをよく見ている（常連として）
- 主人公の日常の疲れ、弟の電話、財布が薄いこと
- これらを証人が「外から」描写する
- **重要**：主人公の内面は証人には見えない。行動と外見のみ。

### 幕3：善行の瞬間（証人が目撃する）
- **このパターン最重要の場面**
- 証人は主人公が老人に傘と肉まんを渡す場面を目撃する
- 証人の解釈：「なぜあの子はそんなことをしたのか？　わからない」
- **証人には主人公の内面が見えない**：だから読者にも見えない
- 証人が「なぜ」と疑問に思うことで、読者も「なぜ」と思う

```
構成例：
「彼女がレジの下から財布を出すのを見た。
 老人に何かを渡した。傘と、何か温かいもの。
 老人が去った後、彼女はレジの前に立って、
 しばらく何もしなかった。
 何を考えていたのか、俺にはわからなかった」
```

### 幕4：証人が悪役の言動を目撃する
- 証人は悪役（黒田）がコンビニや廊下で主人公を追い詰める場面も見ている
- **証人の感情**：怒り・胸がざわつく・何か言いたいが言えない
- 証人も「壁」の前に立っている（証人自身も何らかの制約の中にいる）

### 幕5〜6：主人公の危機（証人は外から見ている）
- 証人は主人公の顔が日に日に疲れていくのを見る
- 直接助けることができない証人の「歯がゆさ」を描く
- この歯がゆさが視聴者の代理感情になる

### 幕7：ツイストの人物（証人には正体が見えている）
- **このパターン最大のトリック**：
  - 証人は老人（ツイストの人物）の「正体」を知っている
  - または、老人が何者かを示す「サイン」に気づいている
  - しかし証人は主人公に言わない（言えない or 言う機会がない）
- 読者は証人の視点から老人を見ているが、証人が「何かを知っている」ことがわかる
- この「知っているが言わない」緊張感がツイスト直前を高める

### 幕8：ツイスト（証人の目の前で起こる）
- 老人が正体を明かす場面を、証人が目撃する
- **証人の感情**：「やっぱりそうだったか」（知っていた）or「え……」（驚き）
- 証人視点だからこそ、主人公の反応が「外から」見える
  - 主人公の膝から力が抜ける瞬間
  - 主人公の手が震える瞬間
  - これを「証人の目」で描くことで、より鮮明になる

### 幕9：恩送り（証人がいなくなる or 証人が動く）
**2パターンから選ぶ：**

**パターンA：証人が去る**
```
証人は翌日からその場所に来なくなった。
どこへ行ったかは知らない。
でも、あの夜に見たことは忘れない。
主人公と老人の間に起きたことが、
俺の中で何かを変えた。
（証人自身も「恩送り」の準備ができた示唆でエンド）
```

**パターンB：証人が動く**
```
今日、コンビニで別の誰かが困っているのを見た。
俺はいつもならそこを通り過ぎる。
でも、足が止まっていた。
あの夜、あの子が老人にしたことを思い出しながら。
```

## 視点設計の技術的注意点

### ツイストを隠す伏線の仕込み方
```
NG（見え見えの伏線）:
「老人の指に、場違いなほど精巧な金の指輪が光っている」
→ 証人が最初からこれを注目している = 読者が気づく

OK（隠れた伏線）:
「老人がスマートフォンを取り出した。画面が一瞬光った。
 機種は俺には判断できなかったが、高そうだとは思った」
→ 証人は気にしていない / 読者も流す / ツイストで「あの時のスマホ」と繋がる
```

### 主人公の内面を「見えない」ままにする技術
```
NG: 「結衣は悩んでいた」（内面の説明）
OK: 「彼女はしばらく動かなかった。
    何を考えているかは、俺にはわからなかった」（外側のみ）
```

## リアリティ・チェックリスト

- [ ] 証人がその場所に「いる理由」が自然か
- [ ] 証人の視点から主人公の内面が「見えない」状態を維持しているか
- [ ] 伏線が1回のみ・目立たない形で埋め込まれているか
- [ ] 証人がツイストの人物の「何か」に気づいているか（でも読者には半分しか伝わらない）
- [ ] 証人の「歯がゆさ」が視聴者の代理感情になっているか
- [ ] 恩送りシーンが証人視点で描かれているか
- [ ] 恩送りシーンが400字以上あるか

## 使用方法

```
/story-pattern-witness-reversal テーマ:[テーマ] 主人公:[主人公設定] 証人:[証人の設定] 舞台:[舞台設定]
```

例：
```
/story-pattern-witness-reversal
テーマ: 貧しい看護学生が、医療費を払えない患者のためにお金を立て替えた
主人公: 21歳女性・看護学生・バイト掛け持ち
証人: 病院の夜間守衛・60代男性・元医師だが訳あって守衛をしている
舞台: 都内病院・深夜の待合室・自動販売機コーナー
```
